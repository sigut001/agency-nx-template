# 🗺️ 11 - Roadmap & Next Steps

Dieses Dokument fasst die strategischen Ziele und die nächsten Schritte zur Perfektionierung der Website-Factory zusammen.

## Kurzfristige Prioritäten (Backlog)

- [ ] **Cookie Banner Finalisierung**: Integration in `App.tsx` und Abschluss der E2E-Tests.
- [ ] **Lizenz-Check**: Script zur automatischen Generierung der `licenses.txt`.
- [ ] **Security Headers**: Finales CSP-Setup in der `firebase.json`.
- [ ] **i18n Prototyp**: Umsetzung der multi-language Struktur in Firestore.

## Mittelfristige Vision

- **Visual Regression Testing**: Screenshots-Vergleich für Layout-Stabilität.
- **Accessibility Automation**: Integration von `pa11y-ci` in die Pipeline.
- **Image Optimization**: WebP Automation und automatische Formatskalierung.

## Langfristige Architektur

- **Terraform Integration**: Vollständige Automatisierung des Firebase-Setups (Auth, DB) ohne manuelle Console-Schritte.
- **Multi-Cloud Disaster Recovery**: Automatisierte Backups außerhalb des Firebase-Ökosystems.
